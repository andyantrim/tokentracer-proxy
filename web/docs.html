Create Model Aliases

Define "Virtual Models" (e.g., gpt-4-prod) and map them to specific providers. You can also specify an optional fallback alias for automatic failover.

Integrate with Your App

TokenTracer provides an OpenAI-compatible endpoint. Simply change your base_url and use your TokenTracer API key.

Python (OpenAI SDK)

from openai import OpenAI

client = OpenAI(
    api_key="your_tokentracer_key",
    base_url="https://your-domain/v1"
)

response = client.chat.completions.create(
    model="your-alias-name", # e.g., 'gpt-4-prod'
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)

JavaScript (OpenAI SDK)

import OpenAI from 'openai';

const openai = new OpenAI({
  apiKey: 'your_tokentracer_key',
            baseURL: 'https://your-domain/v1',
});

const completion = await openai.chat.completions.create({
  model: 'your-alias-name',
  messages: [{ role: 'user', content: 'Hello!' }],
});

console.log(completion.choices[0].message);
